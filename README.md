# 🤖 模型开会 (AI Model Arena) 用户使用手册

欢迎使用 **“模型开会”**！这是一款强大的桌面智能助手，它能让多个顶尖的 AI 模型（如 DeepSeek, Qwen 等）同时为您工作。它们会针对同一个问题分别给出答案，并由一个“裁判模型”进行评判和融合，为您提供最完美的结果。

## 一、 准备工作（首次使用必读）

在打开软件之前，您需要准备两样东西：**“钥匙”**（API Key）和**“通行证”**（Bing Cookie，可选）。

### 1. 获取硅基流动 (SiliconFlow) API Key
这是驱动所有 AI 模型运行的“燃料”。

1.  **注册账号**：访问 [硅基流动官网 (SiliconFlow)](https://cloud.siliconflow.cn/i/j7F36Uco)。
2.  **创建 Key**：
    * 登录后，点击左侧菜单的 **“API 密钥”**。
    * 点击 **“新建 API 密钥”** 按钮。
    * 为密钥起个名字（例如“我的模型软件”），然后点击创建。
3.  **复制 Key**：您会看到一串以 `sk-` 开头的长字符，点击复制并保存好，不要告诉他人。

### 2. 获取 Bing 搜索 Cookie（仅需联网搜索时配置）
如果您希望 AI 能通过必应 (Bing) 搜索最新的新闻资料，建议配置此项，否则可能会搜索失败或结果不准。

1.  **打开浏览器**：使用 Edge 或 Chrome 访问 [cn.bing.com](https://cn.bing.com) 并登录您的微软账号。
2.  **打开开发者工具**：在网页任意位置按键盘上的 `F12` 键（或右键点击页面选“检查”）。
3.  **找到 Cookie**：
    * 在开发者工具顶部菜单找到 **“Network” (网络)** 标签页。
    * 按 `F5` 刷新网页。
    * 在列表中随便点击一个请求（通常是 `cn.bing.com` 开头的）。
    * 在右侧的 **“Request Headers” (请求头)** 区域中，找到 **`Cookie:`** 这一行。
    * 复制 `Cookie:` 冒号后面那一大串字符。

## 二、 快速上手（三步走）

### 第一步：添加 API Key
1.  打开软件，看左上角的 **“SiliconFlow API Key”** 区域。
2.  点击 **“增”** 按钮（小加号图标）。
3.  在弹出的框中粘贴您刚才复制的 `sk-...` 密钥，点击确定。
4.  确保下拉框中显示了您的 Key。

### 第二步：选择选手
1.  在左侧 **“选手模型”** 列表中，勾选您想要参赛的模型。
    * *推荐初学者勾选：`DeepSeek-V3` 和 `Qwen2.5-72B`。*
2.  (可选) 在右侧顶部 **“裁判模型”** 下拉框中，选择一个负责总结的模型（或者选择“🚫 不启用裁判”直接看结果）。

### 第三步：提问并开始
1.  在右下方的 **大输入框** 中输入您的问题（例如：“帮我写一份并在周五下午发送的周报模板”）。
2.  点击底部的 **“开始竞技 (Start Arena)”** 按钮。
3.  稍等片刻，您将在 **“🏆 融合结果”** 标签页看到最终答案。

## 三、 功能详解

### 1. API Key 管理
软件支持管理多个 Key，方便切换。
* **切换 Key**：点击下拉框选择不同的 Key。
* **删除 Key**：选中不需要的 Key，点击右侧的 **“删”** 按钮。

### 2. 选择参赛模型与参数调整
* **勾选模型**：被勾选的模型都会回答您的问题。勾选越多，等待时间可能越长，消耗的 Token 也越多。
* **⚙️ 参数设置**：点击模型右侧的小齿轮图标，可以修改该模型的“性格”：
    * **Temperature (温度)**：数值越高 (如 1.0)，回答越有创意但可能乱编；数值越低 (如 0.0)，回答越严谨保守。
    * **Max Tokens**：限制回答的最大长度。

### 3. 设置裁判模型
裁判的作用是阅读所有选手的回答，指出优缺点，并综合成一个最佳答案。
* **启用裁判**：在右上角下拉框选择一个模型（建议使用能力较强的模型，如 `DeepSeek-V3`）。
* **不启用裁判**：如果您只想看各模型的原始回答，选择 **“🚫 不启用裁判”**。
* **裁判指令**：您可以修改 **“裁判指令 (System Prompt)”** 输入框，告诉裁判您的偏好（例如：“你是一个严厉的老师，请指出代码中的错误”）。

### 4. 🌐 联网搜索功能
当您的问题涉及最新新闻或实时数据时使用。
1.  **设置 Cookie**：点击菜单栏的 **“⚙️ 设置”** -> 在 **“Bing Cookie”** 框中粘贴之前获取的 Cookie -> 保存。
2.  **开启搜索**：点击输入框上方的 **“🌐 联网搜索”** 按钮（按钮变绿即为开启）。
3.  **调整条数**：旁边的数字框可以设置搜索参考前几条结果（默认 5 条）。

### 5. 📎 文件投喂与图片识别
软件支持让 AI “看”文件。
* **添加文件**：点击 **“📎 添加文件”**，支持上传：
    * **图片** (`.jpg`, `.png`)：可以让模型描述图片或提取文字（需选择支持视觉的模型，如 `Qwen-VL`）。
    * **文档** (`.docx`, `.txt`, `.py`, `.md` 等)：让 AI 阅读文档内容并进行总结或问答。
    * *注意：暂不支持 PDF 和 Excel。*
* **移除文件**：选中列表中的文件，点击 **“❌ 移除”**。

### 6. 预设与导出
* **场景预设 (右上方)**：
    * 您可以保存当前的裁判设定和模型组合。点击 **“存”** 保存当前配置，下次直接从下拉框选择即可恢复。
* **提示词预设 (输入框上方)**：
    * 常用的提问（如“润色这段文字”）可以保存。输入文字后点击 **“存”**，下次直接选用。
* **📂 导出结果**：
    * 点击底部的 **“导出结果”**，软件会将当前的问答记录（含融合结果、裁判评价、原始回答）生成一个 `.txt` 文件保存在软件目录下。

## 四、 常见问题解答 (FAQ)

**Q1: 点击“开始”后软件没反应或报错？**
* 检查 API Key 是否正确复制，且账户内有余额（SiliconFlow 注册通常有免费额度）。
* 检查网络连接是否正常。

**Q2: 联网搜索结果显示“未找到有效结果”？**
* 您的 Bing Cookie 可能过期了。请重新登录 Bing 获取新的 Cookie，并在“设置”中更新。
* 或者暂时关闭联网搜索功能。

**Q3: 为什么有的模型无法识别我发的图片？**
* 并非所有模型都长了“眼睛”。请确保勾选了支持视觉的模型（通常名字里带 `VL` 或 `Vision`，例如 `Qwen2-VL` 或 `Llama-3.2-Vision`）。

**Q4: 怎么修改软件界面的颜色？**
* 点击菜单栏 **“⚙️ 设置”**，您可以自定义背景颜色、文字颜色和字体大小，打造您的专属主题。
